# FoxScript
python脚本，数据处理、编码转换。

# http_scan.py
介绍
```
通过请求，检测状态码，将符合状态码的URL存储至文件
```
可自定义的参数：
```
URL的前缀和后缀
返回包状态码
连接超时时长
读取的文本路径
成功结果的文本保存路径
```

# move_some_type_file.py
介绍
```
将指定文件夹下(包括子文件夹)的所有指定后缀文件复制至目标文件夹
```

# substr_md5.py
介绍
```
MD5子串爆破，停止程序时将会保存一个临时文件用于下一次继续爆破
```
自定义参数
```
需要匹配的子串内容
需要匹配的子串初始位置
需要匹配的子串长度
```
# swagger_url.py
介绍
```
提取swagger api-docs中的路径信息
```
自定义参数
```
保存的url文件路径
```
# x_decode.py
介绍
```
对/x开头数据进行解码
```
